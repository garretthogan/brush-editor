# GBrush Editor

Brush editor and maze generator.

## Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

**One-time setup** (if you haven’t already):

1. Push this repo to GitHub.
2. In the repo: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.

**Deploy**

- **Automatic:** Push to the `main` branch. The workflow builds and deploys.
- **Manual:** In the repo go to **Actions → “Deploy to GitHub Pages” → Run workflow**.

The site will be at `https://<your-username>.github.io/brush-editor/` (or your repo name).
